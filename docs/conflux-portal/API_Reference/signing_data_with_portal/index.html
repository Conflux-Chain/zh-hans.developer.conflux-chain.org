使用 ConfluxPortal 浏览器插件钱包对数据签名

签名方法##

ConfluxPortal 是由 MetaMask 继承而来，MetaMask 中有多种签名方法，您可以在 MetaMask 文档中查阅所有签名方法。

ConfluxPortal 中有何改善##

  • 我们使用personal_sign的逻辑来优化cfx_sign方法。并且我们推荐使用 cfx_sign方法代替 personal_sign方法。
  • 在这里signTypedData, signTypedData_v1, signTypedData_v3和signTypedData_v4的原理仍然和 MetaMask 一样。但是我们计划在主网上线前对其进行优化。
